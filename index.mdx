## What is Claw Code?

Claw Code (`claw`) is an open-source, independent implementation of a terminal-based AI coding agent. It provides:

- **Interactive REPL** — A persistent shell session with slash command support, tab completion, and live streaming output
- **One-shot mode** — Run a single prompt non-interactively and exit, perfect for scripts and CI pipelines
- **Rich tool suite** — Execute bash commands, read/write/edit files, search codebases, fetch web pages, and more
- **Session persistence** — Save and resume conversations as JSON files
- **Multi-provider support** — Works with Anthropic's Claude, any OpenAI-compatible endpoint, and xAI/Grok

<Note>
  Claw Code is an independent open-source project, not the official Anthropic Claude Code product. It is not affiliated with or endorsed by Anthropic.
</Note>

## Get up and running

<Steps>
  <Step title="Install Rust and build">
    Install the stable Rust toolchain, then build and install the `claw` binary from source.

    ```bash
    cd rust
    cargo install --path crates/claw-cli --locked
    ```
  </Step>
  <Step title="Set your API key">
    Export your provider credentials. Anthropic is the default:

    ```bash
    export ANTHROPIC_API_KEY="sk-ant-..."
    ```
  </Step>
  <Step title="Start the interactive shell">
    Run `claw` to open the interactive REPL and start coding with AI assistance:

    ```bash
    claw
    ```
  </Step>
  <Step title="Run your first prompt">
    Type a prompt or use `/help` to explore available commands:

    ```bash
    claw prompt "summarize this workspace"
    ```
  </Step>
</Steps>
