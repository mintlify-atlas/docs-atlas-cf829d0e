## Prerequisites

Before you begin, make sure you have the following:

- **Rust stable toolchain** — Install from [rustup.rs](https://rustup.rs). Run `rustup update stable` if you already have Rust to ensure you are on a stable release.
- **API credentials** — A key for at least one supported provider (Anthropic, OpenAI-compatible, or xAI/Grok). See [Authentication](/authentication) for details.

## Installation

<Steps>
  <Step title="Clone the repository">
    Clone the Claw Code repository to your local machine:

    ```bash
    git clone https://github.com/soongenwong/claudecode
    cd claudecode
    ```
  </Step>
  <Step title="Build the release binary">
    Navigate to the `rust/` workspace and build the `claw-cli` crate in release mode:

    ```bash
    cd rust && cargo build --release -p claw-cli
    ```

    The compiled binary is placed at `rust/target/release/claw`.
  </Step>
  <Step title="Install globally">
    Install the binary to your Cargo bin directory so it is available on your `PATH`:

    ```bash
    cargo install --path crates/claw-cli --locked
    ```

    Run this command from inside the `rust/` directory. The `--locked` flag ensures dependencies match the checked-in `Cargo.lock`.
  </Step>
  <Step title="Set your API key">
    Export your API key as an environment variable. Anthropic is the default provider:

    ```bash
    export ANTHROPIC_API_KEY="sk-ant-..."
    ```

    For other providers, see [Authentication](/authentication).
  </Step>
  <Step title="Run Claw Code">
    Start the interactive shell:

    ```bash
    claw
    ```

    You should see the Claw Code REPL prompt. Type a message or `/help` to explore available commands.
  </Step>
</Steps>

## First commands

Once `claw` is installed, try these commands to get oriented:

**Open the interactive shell**

```bash
claw
```

Starts a persistent REPL session. Use slash commands like `/help`, `/status`, and `/exit` to navigate.

**Run a one-shot prompt**

```bash
claw prompt "summarize this workspace"
```

Sends a single prompt to the model and exits. Useful in scripts and CI pipelines.

**Resume a previous session**

```bash
claw --resume session.json /status
```

Reloads a saved session from a JSON file and continues where you left off.

**Show all options**

```bash
claw --help
```

Prints the full command reference, including flags for `--model`, `--permission-mode`, `--output-format`, and all subcommands.

<Tip>
  Run `claw --help` for the complete command list, including agents, skills, and system-prompt flows. The default model is `claude-opus-4-6`, which you can override with `--model`.
</Tip>
